«Автоматизированная система «Клавиатурный тренажер» с функциями администратора»

В этом разделе справки описан интерфейс для обучающегося: вход в систему, выбор упражнения, выполнение упражнения и просмотр личной статистики.

Работа в тренажёре обычно выглядит так: пользователь входит в систему → выбирает уровень сложности и упражнение → выполняет упражнение → затем анализирует результат в статистике.

Назначение: эти страницы используются для доступа к функционалу тренажёра.

Раздел «Вход» предназначен для пользователей, у которых уже есть учетная запись. Раздел «Регистрация» — для создания новой учетной записи обучающегося.

  • Вход — ввод данных учетной записи и переход в раздел «Тренажёр».
  • Регистрация — заполнение формы и создание учетной записи.
  • После успешного входа пользователь получает доступ к страницам: «Выбор упражнения», «Выполнение упражнения», «Статистика».

Где находится: раздел «Тренажёр» → «Выбор упражнения».

Страница предназначена для выбора уровня сложности и конкретного упражнения. Упражнения отображаются карточками.

Основные элементы страницы:

  • Выбор уровня сложности (1–5) — влияет на набор доступных упражнений.
  • Список упражнений — набор карточек выбранного уровня. Номер упражнения на карточке соответствует его позиции в списке упражнений данного уровня.
  • Переход к выполнению — при выборе карточки открывается страница «Выполнение упражнения».
  • Если на выбранном уровне нет упражнений — отображается заглушка «Нет упражнений для этого уровня».

Где находится: открывается после выбора упражнения.

Страница предназначена для тренировки набора текста. Пользователь вводит символы последовательно, ориентируясь на показанный текст и визуализацию клавиатуры.

Основные элементы страницы:

  • Текст упражнения — символы, которые нужно набрать в указанном порядке.
  • Визуализатор клавиатуры — подсветка зон/клавиш, помогает ориентироваться во время набора.
  • Показатели выполнения — скорость (зн/мин), процент ошибок и время выполнения.
  • Завершение упражнения — после окончания прохождения результат сохраняется и позже доступен на странице «Статистика обучающегося».

Где находится: раздел «Тренажёр» → «Статистика».

Страница предназначена для анализа результатов прохождения упражнений. Статистика отображается отдельно для каждого уровня сложности.

Что отображается на странице:

  • Выбор уровня сложности (1–5) — показываются результаты только выбранного уровня.
  • Графики — два графика: среднее время нажатия на клавишу и процент ошибок. По оси X отображается номер упражнения в списке упражнений выбранного уровня.
  • Если для выбранного уровня нет результатов — отображается заглушка «Нет данных для этого уровня».
